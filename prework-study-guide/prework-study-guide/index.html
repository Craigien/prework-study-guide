HTML

  • The head element contains information about the webpage such as the following:
    • Meta tags containing metadata
    • The title that will appear on the browser tab
  • The body element represents the visible content shown to the user such as the following:
    • Headings from h1 to h6
    • Lists both ordered and unordered
    • images and links
    • paragraphs
    • page breaks

CSS

  • CSS can be used three different ways
    • Inline CSS
    • Internal CSS
    • External CSS
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be assigned with color names or by using hexidecimal.
  • The * wildcard will assigned properties to all elements.
  • A class attribute can be created by using a . followed by the class name.

Git

  • git commands
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • There are a number of different data types such as:
    • Strings
    • Numbers
    • Bools
  • An array is a type of data structure that holds a group of data that is related in some way.
  • Control flow is the order in which a computer executes code in a script.
    • In JavaScript, code is executed from top to bottom.
